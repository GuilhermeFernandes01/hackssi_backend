## Variáveis Locais

Variáveis locais utilizadas para rodar e debugar a aplicação
```

SWAGGER=true
MONGO_URI=$MONGO_URI
MONGO_DB_DATABASE=$MONGO_DATABASE
USERS_COLLECTION=$MONGO_USER_COLLECTION
ENVIRONMENT=dev
```
